# skills

Claude Code 用 アジャイル開発スキル集（mrtry-lab 配布版）。

詳細な使い方とインストール手順は本格版 README で記載予定。

## インストール

```bash
gh skill install mrtry-lab/skills <skill-name> --agent claude-code --scope user
```

## 同梱スキル

- agile-product-vision
- agile-epic
- agile-create-backlog
- agile-refine-backlog
- agile-story-to-task
- agile-task-implementation
- create-issue
- create-pull-request
